myELIZA
=======

Group Members:
- [Your Name]

Description:
A command-line C++ recreation of ELIZA. The program plays the role of a
therapist, using regex-based pattern matching across topic categories
(generic verbs, love/relationships, concern, financial, wellbeing,
education, entertainment, and technology & social media) to generate
responses to free-form natural-language input.

Compilation:
    make
    (or manually: g++ -std=c++17 -Wall -Wextra src/*.cpp -o myeliza)

Running:
    ./myeliza

Type "bye" at any prompt to end the session.
